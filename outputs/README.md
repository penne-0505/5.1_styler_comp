# 出力ファイル (Output Files)

このディレクトリには、各スタイラーによる出力結果を格納します。

## ディレクトリ構成

```
outputs/
├── styler_a/
│   ├── output_001.txt
│   ├── output_002.txt
│   └── ...
├── styler_b/
│   ├── output_001.txt
│   ├── output_002.txt
│   └── ...
└── styler_c/
    ├── output_001.txt
    ├── output_002.txt
    └── ...
```

## ファイル命名規則

- 各スタイラーのサブディレクトリ内に出力ファイルを配置
- 出力ファイル名は対応する入力ファイル名と一致させる
  - 例: `inputs/input_001.txt` → `outputs/styler_a/output_001.txt`
- この命名規則により、同じ入力に対する異なるスタイラーの出力を簡単に比較できます

## スタイラーの追加

新しいスタイラーを追加する場合：
1. `outputs/` 配下に新しいディレクトリを作成（例: `styler_d/`）
2. 各入力ファイルに対応する出力ファイルを配置
3. `comparisons/` ディレクトリに比較結果を記録

## 使用方法

1. LLMにスタイラーを適用して出力を生成
2. 生成された出力を対応するスタイラーディレクトリに保存
3. ファイル名は `output_XXX.txt` の形式（XXXは入力ファイルの番号と一致）
