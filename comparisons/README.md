# 比較結果 (Comparison Results)

このディレクトリには、異なるスタイラー間の出力比較結果を格納します。

## ディレクトリ構成

比較結果は以下の形式で整理することを推奨します：

```
comparisons/
├── comparison_001.md    # input_001に対する全スタイラーの比較
├── comparison_002.md    # input_002に対する全スタイラーの比較
├── summary.md           # 全体的な比較サマリー
└── analysis/            # 詳細な分析レポート
    ├── quality_analysis.md
    ├── style_differences.md
    └── recommendations.md
```

## 比較の観点

各比較ファイルには以下の観点を含めることを推奨します：

1. **正確性**: 出力内容の正確さと妥当性
2. **スタイル**: 文体、トーン、表現方法
3. **完全性**: 質問に対する回答の網羅性
4. **可読性**: 理解しやすさ、構造の明確さ
5. **実用性**: 実際の使用における有用性
