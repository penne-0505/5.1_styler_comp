# 入力ファイル (Input Files)

このディレクトリには、LLMスタイラーのテストに使用する入力例（設問）を格納します。

## ファイル命名規則

- `input_001.txt` - 最初のテスト入力
- `input_002.txt` - 2番目のテスト入力
- など...

各入力ファイルには、LLMに送信するプロンプトやテキストを記述してください。

## 使用方法

1. 新しいテストケースを追加する際は、連番でファイルを作成してください
2. ファイル名は `input_XXX.txt` の形式を推奨します（XXXは3桁の連番）
3. 各入力ファイルには、テストしたい具体的な内容を記述してください

## サンプル

以下のようなテストケースの例：
- プログラミング関連の質問
- 文章の要約依頼
- クリエイティブな作文依頼
- 技術的な説明依頼
